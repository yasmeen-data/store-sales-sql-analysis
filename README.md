# store-sales-sql-analysis

## Project overview
Beginner SQL project analyzing store sales data using PostgreSQL.
The goal of this project is to generate business insights from store sales using SQL queries.

# Database Structure
This project uses a single table:

# SALES
1. order_id
2. customer_name
3. city
4. order_date
5. amount

# SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- Aggregation Functions (SUM, AVG, COUNT, MAX, MIN)
- EXTRACT
- NULL handling
- Aliases

## Business Questions Answered
1. What is the total revenue?
2. What is the total revenue per city?
3. Which city generated the highest revenue?
4. Who is the top spending customer?
5. Which customers spent more than 1000?
6. Which month generated the highest revenue?
7. Which city has the highest average order value?
8. How many orders were placed in January?

## Key Insights
* Mumbai generated the highest revenue.
* Asha is the top spending customer.
* January generated more revenue than February.
* Delhi has the highest average order value.
